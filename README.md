wjet => Webix Jet CLI tools
============================


[![npm version](https://badge.fury.io/js/wjet.svg)](https://badge.fury.io/js/wjet)


### How to install

```bash
npm install -g wjet
```

or 

```bash
yarn global add wjet
```

### How to use

- Create new app

```bash
mkdir myApp
cd myApp
wjet init
```

- Add feature

```bash
wjet add feature
```

- Add widget

```bash
wjet add widget
```

- New view wizard

```bash
wjet add view
```

- New model wizard

```bash
wjet add model
```
