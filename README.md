wjet => Webix Jet CLI tools
============================


[![npm version](https://badge.fury.io/js/wjet.svg)](https://badge.fury.io/js/wjet)


### How to install

```
npm install -g wjet
```

or 

```
yarn global add wjet
```

### How to use

- Create new app

```
mkdir myApp
cd myApp
wjet init
```

- Add feature

```
wjet add feature
```

- Add widget

```
wjet add widget
```
